<h2>🕶NEOSHADE<br></h2>
쇼핑몰의 기본적인 기능을 포함하여 제품검색, 장바구니, 결제 등을 구현하였습니다.<br>
<br>
<h3>1. 기술 스택<br></h3>
Java: 11<br>
Spring Boot: 2.7.10<br>
database: MariaDB<br>
build: Gradle<br>
